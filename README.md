# VS Code Pack — viamigo (Node)

## Quick start
1. Copy this folder into your project root.
2. Copy `.env.example` to `.env` and set variables.
3. Open in VS Code (optionally **Reopen in Container**).
4. Press **Run and Debug** → **Node: npm run dev** or run task **Dev server**.

## Notes
- Port **3000** is forwarded in the dev container.
- ESLint/Prettier recommended; wire scripts in `package.json` if not present.
